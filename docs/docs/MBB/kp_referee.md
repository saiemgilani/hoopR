# `kp_referee`

## Description

Get referee game log


## Usage

```r
kp_referee(referee = 714363, year = 2021)
```


## Arguments

Argument      |Description
------------- |----------------
`referee`     |     Referee ID
`year`     |     Year of data to pull


## Value

A data frame with 7 columns

## Examples

```r
kp_referee(referee = 714363, year = 2021)
```
